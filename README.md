## Scrum 新手村 關卡資訊

> 本週題目主要內容是介紹 Scrum 工作法的知識點，UI 設計師可以設計關卡內容與網頁互動效果，透過頁面關卡來呈現知識點，讓挑戰者藉由你設計的內容更了解 Scrum。

### 關卡需包含以下知識點

- 角色介紹（PO、SM、開發）
- 產品待辦清單（Product Backlog）
  - 需加入 Jira 介紹、LOGO
- Sprint Planning
- 短衝清單
- Sprint Point
- Sprint 流程
  - Daily
  - Review（適合使用 Confluence）
  - Retro（適合使用 Confluence）
